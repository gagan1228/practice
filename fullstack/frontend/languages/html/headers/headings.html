<H1> is an HTML tag that identifies the first heading on a page. It is used as the name of the page or publication.

An <H2>tag marks the first sub-heading after your document's main heading. It defines the second-level headings on your webpage.

<H3> tag defines the third level heading in the HTML document.

<H4> element defines a level-four heading. This heading is more important than an H5 but less important than an H3.

<H5> element defines a level-five heading.
<H6>defines the least important heading.
